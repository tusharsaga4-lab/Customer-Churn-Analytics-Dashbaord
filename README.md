# AI-Powered Telecom Customer Churn Analytics Dashboard

## Overview
This project analyzes telecom customer churn using Python, Pandas, Power BI, DAX, and Power Query.

## Tools Used
- Python
- Pandas
- Power BI
- DAX
- Power Query

## Key Insights
- Overall Churn Rate: 26.6%
- Fiber Optic customers have the highest churn rate (41.9%)
- Electronic Check users have the highest churn rate (45.3%)
- Most churn occurs within the first year
- Month-to-Month contracts contribute the highest churn

## Dashboard Pages
1. Executive Overview
2. Customer Risk Analysis
3. Business Recommendations

- Promote annual contracts
- Improve Fiber Optic service quality
